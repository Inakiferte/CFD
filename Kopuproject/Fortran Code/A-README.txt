Bertsio honetan aurrerapen handia egin dugu. Izatez, planteamendu orokorra egokia zen baina akatsak aurkitu genituen.
Alde batetik, F eta G-ri dagokionez, notazio desegokia erabili genuen. Bestalde, irudi grafikoa eratzeko inplementatutako metodoa ere ez zen guztiz egokia. Izan ere, x osagaieko eta y osagaieko bektoreak gehitzeko orduan, bakoitzari dagokion matrizea dimentsioa ezberdinekoa izanik, matrize berria sortzeko metodoa gaizki zegoen. Kasu honetan ere, nahiz eta ideia egokia izan, notazioa ez zen zuzena. Horretaz gain, mugalde baldintzei dagokienez, paretei dagokien posizioko abiadura nulua dela aplikatu dugu.
